# JulyGuild（V2.0.1）

> 本贴使用 [MCBBS Markdown To BBCode Converter](https://mm2bc.ustc-zzzz.net/) 生成

## 插件特点

* 开源
  
* 完美的 GUI 支持
* 高度自定义
  
* 多版本支持

## 功能

* 加入公会

* 公会信息

* 个人信息

* 传送到公会主城

* 公会排行榜

* 入会审批

* 成员管理

* 捐赠

* 公会商店

* 设置公会主城

* 公会升级

* 公会图标

* 全员集结令

* 图标仓库
